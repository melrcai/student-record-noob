## Description
This is technically a simple student record manager commonly used to add, view, delete, and sort student records. There’s nothing too special about it — I built this as my first beginner project to familiarize myself with the C programming language.
To my future self: there’s probably not much to see here, and you’re likely giggling while reading this. A little cringe, but it marks the beginning of this hella journey XD.

---

## Features
- Add student records
- View all students
- Delete student records
- Sort student data
- File saving and loading

---

## How It Works
The program uses a menu-based interface displayed in the terminal.  
Users select an option by entering a number corresponding to an action.  
Student data is stored in a file so records remain saved even after closing the program.
